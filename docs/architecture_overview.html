Agentic Vibe Fleet 🛸

A context-first framework for high-integrity autonomous engineering.

📜 The Constitution

Located in GEMINI.md. Defines non-negotiable procedures: Plan-First, Context-First, and Cost Gating. It is the long-term memory for all sessions.

🗺️ Architecture Map

Located in Project_Structure.md. The source of truth for all files. If it's not in the changelog, it's not in the project.

🛠️ Agentic Skills

Located in scripts/. Autonomous tools for hygiene: Verify (Structure), Optimize (Logs), Update (Docs), and Generate (Prompts).

✅ Contract Testing

Located in bruno/. Contract-first API validation. No code is committed without passing Bruno tests or an explicit exception.

🚀 The Vibe Coding Workflow

1
Define intent in English and run generate_bootstrap_prompt.py.
2
Copy the generated prompt into a fresh Agent Mode session.
3
Gemini analyzes Patterns and Structure to build a surgical implementation plan.
4
Execute changes, run verify_structure.py, and validate with Bruno.
5
Run optimize_changelog.py to keep history clean for the next session.
